CUPW & CPAA Wage Increase

The CUPW and CPAA wage increase are dollar impacts due to negotiated wage increases for the CPAA and CUPW unions.

They are calculated for all Hours Related dollar impacts, as well as the C&D Route (over assessed) and Rest&BG (rest and boot and glove) which are Non Hours Related (see the Hours and Non Hours Related Split).

The C&D and MP ActGroups are only CUPW employees (Union = CUPW) and the CO ActGroup is made up of a mix of CUPW and CPAA (Union = CUPW/CPAA). This split, along with the weighted wage increase for each SHN is found in the T007 table, which can be accessed via the Labour Type Rate Analysis report.

An abridged example of the T007 and a few examples of the rate is calculated is shown below.

T007 - Wage Increase % (Example)
GM Union P01 P02 ... P12 CUPW01 CUPW02 ... CUPW12 CPAA01 CPAA02 ... CPAA12
SHN103979 CUPW 0.01 0.01 0.01 1 1 1 0 0 0
SHN103979 CUPW/CPAA 0.018 0.018 0.018 0.2 0.2 0.2 0.8 0.8 0.8
SHN103980 CUPW 0.01 0.01 0.01 1 1 1 0 0 0
SHN103980 CUPW/CPAA 0.0184 0.0184 0.0184 0.16 0.16 0.16 0.84 0.84 0.84

The cells P01, P02 ... P12 are the weighted wage increase for each SHN/Union combination. They are calculated outside of the DLM.

The cells CUPW01, CUPW02 ... CUPW12 and CPAA01, CPAA02 ... CPAA12 are the percentages of each union type amongst the employees in each SHN/Union combo, for CUPW and CPAA respectively.

There are monthly CPAA and CUPW wage increases which are no directly shown in the T007, but are used to calculate the weighted wage increases. An exmaple wage increase table, which was used to calculate the example T007 above, is shown below.

CUPW & CPAA Wage Increases (Example)
Union P01 P02 ... P12
CUPW 0.01 0.01 0.01
CPAA 0.02 0.02 0.02

General Formula

Weighted Return = [CUPW Employee %] * [CUPW Wage Increase (%)] + [CPAA Employee %] * [CPAA Wage Increase (%)]
CUPW Wage Increase = ([TY Dollars] - [TY Dollars] / (1 + [Weighted Return])) * [CUPW Employee %]
CPAA Wage Increase = ([TY Dollars] - [TY Dollars] / (1 + [Weighted Return])) * [CPAA Employee %]

Example 1:
P01; SHN = SHN103979; ActSubGrp = LC (Union = CUPW)

From Table: CUPW Wage Increase = 1%; CPAA Wage Increase = 2%; CUPW Employee % = 100%; CPAA Employee % = 0%
From BW 9000: TY Dollars = 100

P01 Weighted Wage Increase = (1)*(0.01) + (0)*(0.02) = 0.01
CUPW Wage Increase = [100 - 100 / (1 + 0.01)] * (1) = $0.99
CPAA Wage Increase = [100 - 100 / (1 + 0.01)] * (0) = $0.00

Example 2:
P01; SHN = SHN103980; ActSubGrp = CO (Union = CUPW/CPAA)

From Table: CUPW Wage Increase = 1%; CPAA Wage Increase = 2%; CUPW Employee % = 20%; CPAA Employee % = 80%
From BW 9000: TY Dollars = 100

P01 Weighted Wage Increase = (0.16)*(0.01) + (0.84)*(0.02) = 0.0184
CUPW Wage Increase = [100 - 100 / (1 + 0.0184)] * (0.16) = $0.29
CPAA Wage Increase = [100 - 100 / (1 + 0.0184)] * (0.84) = $1.52
