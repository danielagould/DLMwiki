GLCE Group (Dollars)

The GLCE Group is the hour type for each hour work, whether Full Time (FT), Part Time (PT), Temporary (TMP) or Overtime (OT).

Each entry in the Dollars raw data input (see Dollars) has a cost element code. These cost element codes exist in a hierachy that the DLM uses to map each entry.


The DLM extracts this hierarchy on a periodic basis creates a list of parent and child nodes, effectively mapping the lowest Cost element Codes to the highest Cost Element Groups (e.g. Labour Excl Benefits). This is list is stored in the tblHierarchies table (DLM_Inputs).

Using the parent/child list and the Cost Element to GLCEGrp table tblGLCEGrp (see below) the DLM maps all cost elemets to a GLCE Group.

For the dollars the DLM uses additional GLCE Groups as compared with the hours mapping. This is because the dollars are separated into Hours Related vs Non Hours Related) (although all are Labour Related).

Hours Related are mapped to FT, PT, TMP and OT.

Non Hours Related are mapped to HH Bonus, NonHrs, OthAllw, OthLab, Rest&BG and Route
