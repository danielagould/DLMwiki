Time Group (Hours)

The BW hours data contains a Time Type Code and Time Type Name. These Time Types are rolled up into a hierarchy of Time Types, which BW uses to organize hours recorded.


The DLM performs a similar function and maps all Time Type Codes in BW to a Time Group. The process to build this mapping is complex and data used is pulled directly from SAP on a periodic basis. Ultimately the mapping is based off the table below (tblTimeCode) in the DLM, which uses the higher levels of the Time Type hierarchy to map all Type Type Codes to a Time Group (TimeGrp). It also maps all Hours Worked to FT/PT/OT/TMP.

tblTimeCode
Time Type TimeCodeSeq TimeCode TimeGrpSeq TimeGrp
Full Time Hours 1 FT 1 Hr WK
Part Time Hours 2 PT 1 Hr WK
Term/Temporary Hours 3 Tmp 1 Hr WK
Overtime Hours 4 OT 1 Hr WK
Other Paid Attendance 5 Train/Oth 2 Train/Oth
Absenteeism (FT&PT) 6 PdAbs 3 PdAbs
Absenteeism - Other (FT & PT) 6 PdAbs 3 PdAbs
Other Absenteeism (Term and Term/Temp) 6 PdAbs 3 PdAbs
Vacation 7 Vacation 4 Vacation
Statutory Holiday 8 Stat 5 Stat
