Activity Groups & Sub Groups

The Activity Type is a code in BW that assigns each hour recorded in the 9502 to a specific work type. These include Mail Processing, Letter Carrying, etc. The DLM takes these ActTypes and maps them to ActGroups (Mail Processing, Collection & Delivery and Counter Operations) and ActSubGroups. ActSubGroups only applies to Collection and Delivery which is divided into Letter Carriers (LC), Letter Carrier Assisstants (LCA) and Mail Service Couriers (MSC).
